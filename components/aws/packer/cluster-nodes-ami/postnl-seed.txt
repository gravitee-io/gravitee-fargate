# The PostNL Cluster OS image : PXEless boot of the cluster

### Buildnd the AWS AMI for cluster nodes

* In your environement, you need :
  * `pulumi`
  * `aws-cli`
  * `packer`
  * the `AWS` credentials for the tenant you work for, on the lcal filesystem at `~/.aws/credentials`

* Execute :

```bash
# ---
./components/aws/packer/cluster-nodes-ami/packer-quick-install.sh
./components/aws/packer/cluster-nodes-ami/build-ami.sh

packer build -var 'ami_name_prefix=http-benchmarking' template.json
```
